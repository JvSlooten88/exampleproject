src.subpackage.functions module¶

This is a description of the functions.py module. This is also parsed by the Sphinx apidoc tool, and it’s responsive to all ReST syntax/gimmicks.

For example, you make make notes:

Note

And warnings:

Warning

This is a warning.

Now, you can simply add docstrings to your functions/classes, and they will turn up in your documentation when processed with sphinx-apidoc.

Note

You should use Numpy-docstrings instead of the regular Python docstrings. When using Numpy-docstrings, make sure to use the Sphinx-extension “napoleon” (sphinx_ext_napoleon), which is the engine to parse Numpy-docstrings!
